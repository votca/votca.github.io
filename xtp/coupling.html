coupling¶

Compute the electronic couplings from log and orbital files. The following table contains the defaults input options for the calculator,

Property Name

Default Value

Description

Valid Input

dftpackage.package.name

xtp

Name of the DFT package
xtp
orca

dftcoupling_options.degeneracy

0.0

Criterium for the degeneracy of two levels

float+

dftcoupling_options.levA

1
Output HOMO, …, HOMO-levels; LUMO, …, LUMO+levels,
molecule A

int+

dftcoupling_options.levB

1
Output HOMO, …, HOMO-levels; LUMO, …, LUMO+levels,
molecule B

int+

moleculeA.log

A.log

Log file of molecule A

moleculeA.orbitals

A.orb

Orbitals file

moleculeB.log

B.log

Log file of molecule B

moleculeB.orbitals

B.orb

Orbitals file

dimerAB.log

AB.log

Log file of dimer AB

dimerAB.orbitals

A.orb

Orbitals file

Note

An xml file containing the defaults for the coupling calculator can be found at $VOTCASHARE/xtp/xml/coupling.xml
