
NAME
****
**csg_call postadd average** - Part of the VOTCA package

SYNOPSIS
********
**csg_call postadd average** [OPTIONS]

**csg_call postadd average** [--help]

Usage: postadd_average.sh

DESCRIPTION
***********
postadd average script,
calcs averages of (${name}.DIST.cur) for the past few steps
and saves it to ${name}.DIST.avg
DIST can be specified by average.what option
