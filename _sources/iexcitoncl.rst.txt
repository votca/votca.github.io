
iexcitoncl
**********
Classical coupling of transition charges
The following table contains the defaults input options for the calculator,

.. list-table::
   :header-rows: 1
   :widths: 30 20 15 15
   :align: center

   * - Property Name
     - Description
     - Default Value
     - Valid Input
   * - map_file
     - xml file with segment definition
     - votca_map.xml
     - 
   * - job_file
     - name of jobfile to which jobs are written
     - exciton.jobs
     - 
   * - use_states
     - Use pair of segments and exciton labels
     - false
     - bool
.. note::
   An *xml* file containing the defaults for the `iexcitoncl` calculator can be found at `$VOTCASHARE/xtp/xml/iexcitoncl.xml`
