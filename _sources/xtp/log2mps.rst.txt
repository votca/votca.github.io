
log2mps
*******
Generates an mps-file (with polar-site definitions) from a QM log-file
The following table contains the defaults input options for the calculator,

.. list-table::
   :header-rows: 1
   :widths: 30 20 15 15
   :align: center

   * - Property Name
     - Default Value
     - Description
     - Valid Input
   * - package
     - orca
     - QM package
     - orca
.. note::
   An *xml* file containing the defaults for the `log2mps` calculator can be found at `$VOTCASHARE/xtp/xml/log2mps.xml`
