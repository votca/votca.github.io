
neighborlist
************
Determines neighbouring pairs for exciton transport
The following table contains the defaults input options for the calculator,

.. list-table::
   :header-rows: 1
   :widths: 30 20 15 15
   :align: center

   * - Property Name
     - Default Value
     - Description
     - Valid Input
   * - segments.type
     - 
     - segments names if the segmentname method is used
     - 
   * - segments.cutoff
     - 
     - cutoff value if the segmentname method is used
     - 
   * - cutoff_type
     - constant
     - method to define the cutoff
     - | constant
       | segmentname
   * - constant
     - 1.5
     - | cutoff for all other types of pairs if the constant method
       | is used
     - float+
   * - use_exciton_cutoff
     - false
     - Use exciton cutoff
     - bool
   * - exciton_cutoff
     - 
     - cutoff for classical exciton transition charge treatment
     - 
.. note::
   An *xml* file containing the defaults for the `neighborlist` calculator can be found at `$VOTCASHARE/xtp/xml/neighborlist.xml`
