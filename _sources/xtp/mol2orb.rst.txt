
mol2orb
*******
Converts molden files to orb files.
The following table contains the defaults input options for the calculator,

.. list-table::
   :header-rows: 1
   :widths: 30 20 15 15
   :align: center

   * - Property Name
     - Default Value
     - Description
     - Valid Input
   * - basisset
     - def2-tzvp
     - Basis set for MOs
     - 
   * - auxbasisset
     - aux-def2-tzvp
     - Auxiliary basis set for RI
     - 
.. note::
   An *xml* file containing the defaults for the `mol2orb` calculator can be found at `$VOTCASHARE/xtp/xml/mol2orb.xml`
