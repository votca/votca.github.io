
orb2fchk
********
Converts orb files to gaussian fchk files.
The following table contains the defaults input options for the calculator,

.. list-table::
   :header-rows: 1
   :widths: 30 20 15 15
   :align: center

   * - Property Name
     - Default Value
     - Description
     - Valid Input
   * - qmstate
     - n
     - QMstate in string format, e.g. n, s4 etc.
     - 
   * - diff2gs
     - false
     - For excited states output difference to groundstate
     - bool
.. note::
   An *xml* file containing the defaults for the `orb2fchk` calculator can be found at `$VOTCASHARE/xtp/xml/orb2fchk.xml`
