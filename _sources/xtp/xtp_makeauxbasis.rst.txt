
NAME
****
**xtp_makeauxbasis** - Part of the VOTCA package

SYNOPSIS
********
**xtp_makeauxbasis** [OPTIONS]

**xtp_makeauxbasis** [--help]


DESCRIPTION
***********
Creates votca xml aux-basissetfiles from votca basissetfiles

OPTIONS
*******
::

  optional arguments:
    -h, --help                           show this help message and exit
    -f BASISFILE, --basisfile BASISFILE  xtp basissetfile
    -o OUTFILE, --outfile OUTFILE        optional file to write basisset to
    -g GROUPING, --grouping GROUPING     Cutoff at which basisfunctions are
                                         grouped together in deviation from the
                                         arithmetic mean; Default 0.1
    -c CUTOFF, --cutoff CUTOFF           Cutoff for very localised
                                         basisfunctions; Default 60
    -e ELEMENT, --element ELEMENT        Print out only the element specified
    -l LMAX, --lmax LMAX                 maximum angular momentum in aux
                                         basisset: Default:4
