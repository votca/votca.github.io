
apdft
*****
Write Electrostatic potentials at core positions
The following table contains the defaults input options for the calculator,

.. list-table::
   :header-rows: 1
   :widths: 30 20 15 15
   :align: center

   * - Property Name
     - Default Value
     - Description
     - Valid Input
   * - grid
     - medium
     - grid quality
     - | xcoarse
       | coarse
       | medium
       | fine
       | xfine
   * - state
     - N
     - QM state to generate potential
     - 
.. note::
   An *xml* file containing the defaults for the `apdft` calculator can be found at `$VOTCASHARE/xtp/xml/apdft.xml`
