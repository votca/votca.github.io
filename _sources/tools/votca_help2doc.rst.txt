
NAME
****
**votca_help2doc** - Part of the VOTCA package

SYNOPSIS
********
**votca_help2doc** [OPTIONS]

**votca_help2doc** [--help]


DESCRIPTION
***********
Convert a Votca CLI's help message to RST or GROFF format.



OPTIONS
*******
::

  optional arguments:
  
    -h, --help            show this help message and exit
    -n NAME, --name NAME  Name of the tool to extract the help message
    -o OUT, --out OUT     Output file
    -f {rst,groff}, --format {rst,groff}
                          Output format
    --version             show program's version number and exit
