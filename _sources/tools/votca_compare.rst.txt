
NAME
****
**votca_compare** - Part of the VOTCA package

SYNOPSIS
********
**votca_compare** [OPTIONS]

**votca_compare** [--help]


DESCRIPTION
***********
COMPARES TWO FILES TO EACH OTHER

OPTIONS
*******
::

  optional arguments:
    -h, --help                show this help message and exit
    -f1 FILE1, --file1 FILE1  File1 to compare.
    -f2 FILE2, --file2 FILE2  File2 to compare
    --etol ETOL               Tolerance for floats to be regarded
                              equal,default=1e-5
