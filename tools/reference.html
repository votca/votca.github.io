votca_compare¶

NAME¶

votca_compare - Part of the VOTCA package

SYNOPSIS¶

votca_compare [OPTIONS]

votca_compare [–help]

DESCRIPTION¶

COMPARES TWO FILES TO EACH OTHER

OPTIONS¶

optional arguments:

  -h, --help                show this help message and exit
  -f1 FILE1, --file1 FILE1  File1 to compare.
  -f2 FILE2, --file2 FILE2  File2 to compare
  --etol ETOL               Tolerance for floats to be regarded
                            equal,default=1e-5

votca_help2doc¶

NAME¶

votca_help2doc - Part of the VOTCA package

SYNOPSIS¶

votca_help2doc [OPTIONS]

votca_help2doc [–help]

DESCRIPTION¶

Convert a Votca CLI’s help message to RST or GROFF format.

OPTIONS¶

optional arguments:

  -h, --help            show this help message and exit
  -n NAME, --name NAME  Name of the tool to extract the help message
  -o OUT, --out OUT     Output file
  -f {rst,groff}, --format {rst,groff}
                        Output format
  --version             show program's version number and exit

votca_property¶

NAME¶

votca_property - Part of the VOTCA package

SYNOPSIS¶

votca_property [OPTIONS]

votca_property [–help]

DESCRIPTION¶

votca_property votca_tools, version 2022-dev gitid: c343ca8 (compiled Sep 10 2021, 05:48:21)

Helper for parsing XML files

OPTIONS¶

Allowed options:

  -h [ --help ]            display this help and exit
  --verbose                be loud and noisy
  --verbose1               be very loud and noisy
  -v [ --verbose2 ]        be extremly loud and noisy
  --file arg             xml file to parse
  --format arg           output format [XML TXT]
  --level arg            output from this level
